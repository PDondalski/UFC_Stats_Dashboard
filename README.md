# 🥊 UFC Fight Analysis Dashboard (Streamlit)

This project is an interactive dashboard built with **Streamlit** for exploring UFC fight statistics.  
It allows users to analyze relationships between fighter attributes, performance, and fight outcomes in a simple and visual way.

---

## 📊 Project Overview

The main idea behind this project is to explore UFC fight data and understand how different factors influence fighter performance.

The dashboard is designed to guide the user from general observations to more detailed analysis, with the ability to filter and interact with the data.

---

## 🚀 Features

### 🔍 Data Exploration
- Interactive filters (fighter age)
- Dynamic data table
- Real-time updates based on user input

### 📈 Visualizations
- Scatter plots (comparison of significant strikes)
- Histograms
- Bar charts (distribution of weight classes)

### 🤖 Machine Learning Model
- Linear Regression model predicting number of successful takedowns
- Evaluation metrics:
  - R² score
  - Mean Absolute Error (MAE)
- User input form for predictions

---

## 🧠 Key Insights

- Fighter performance varies across weight classes
- Number of takedown attempts is strongly related to successful takedowns
- The model achieves **moderate predictive performance** (R² ≈ 0.56)
